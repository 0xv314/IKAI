# Accueil





### IKAI – Live-to-Earn (L2E) : un univers Web3 géolocalisé et économique

Ikai est une application mobile Web3 sur **Solana** qui fusionne exploration du monde réel, mécaniques **RPG**, classes interconnectées et **économie décentralisée**. Se déplacer, interagir, accomplir des missions et combattre fait progresser le joueur, lui permettant d’acquérir **ressources**, **objets**, **familiers** et **tokens**. **Concept L2E :** _Vivre, jouer et se déplacer dans le monde réel génère de la valeur tangible._&#x20;

***

### Expérience utilisateur & flux d’entrée

#### 1) Écran d’accueil (Splash)

* **Barre d’énergie arc-en-ciel** à effet néon animé.
* **Logo Slime** animé (référence aux univers _isekai_).
* **Vérification** automatique de la connexion blockchain Solana. Affichage très bref, optimisé mobile.&#x20;

#### 2) Connexion & création de compte

| Méthode                                     | Détails                  | Avantages                                              |   |
| ------------------------------------------- | ------------------------ | ------------------------------------------------------ | - |
| **Wallet Solana** (Phantom, Backpack, etc.) | Connexion Web3 directe   | Accès instantané aux NFT & tokens                      |   |
| **Email**                                   | Authentification **OTP** | Idéal pour débutants ; liaison wallet possible ensuite |   |
| **Téléphone**                               | **OTP** par SMS          | Création rapide, pensée mobile                         |   |

* Assistant intégré si l’utilisateur n’a pas de wallet (création/import en < 1 min + bases de sécurité).
* Les comptes **email/téléphone** sont **évolutifs** : liaison wallet possible à tout moment pour un compte Web3 complet.
* **Récupération :** IKAI envoie **de manière chiffrée par email la phrase secrète** pour permettre, en cas de perte de mobile, de **retrouver l’accès au compte** (éviter la capture d’écran/bout de papier non sûrs).&#x20;

#### 3) Choix du personnage

* **Genre** : Homme · Femme · **Androgyne** (terme corrigé).
* Base visuelle : **3 modèles** par défaut ; **équipements/skins** s’y ajoutent.
* **Avatar/NFT Personnage** : **unique**, **non transférable**, **non vendable**.&#x20;

**3.1) Pack Partenariat (burn)**

* Possesseurs d’un **NFT partenaire** : possibilité de **burn** (wallet dédié) pour obtenir un **NFT IKAI Partenaire** au **visuel unique**.
* Contrairement aux personnages initiaux, ces NFT **sont transférables** (marché secondaire).
* **Burn multiple** → obtenir un NFT plus puissant.
* Exemple “dog” :
  * **1 NFT** burn → pack de base (≈ **0,5 SOL**).
  * **8 NFT** burn → **pack 3** directement, sans avancer de SOL (hors frais TX).&#x20;

#### 5) Attribution du rang

| Groupe de rang    | Rareté               | Animation d’ouverture                   | Impact jeu      |                                                           |
| ----------------- | -------------------- | --------------------------------------- | --------------- | --------------------------------------------------------- |
| **SSS / SS / S**  | **Extrêmement rare** | Effets “pyrotechniques”, holographiques | Impacts majeurs |                                                           |
| **A / B / C**     | **Rare**             | Animations colorées, éclats             | Impacts élevés  |                                                           |
| **D / E / F / G** | **Commun**           | Effets légers ou absents                | Impacts faibles | ([ikai-l2e.gitbook.io](https://ikai-l2e.gitbook.io/ikai)) |

* **Rangs Divins** (12 principaux + demi-divin) : achats spéciaux/événements limités.
* Tout rang peut **upgrader** via **Transcendance** (coût croissant avec le rang).&#x20;

#### 6) Choix de la classe

* **12 classes principales**, chacune liée à une **fonction économique** (inter-dépendance).
* Exemples : **Invocateur** (familiers NFT) · **Agriculteur** (ressources IRL rares pour crafting) · **Guerrier** (combats/donjons ; Ninja/Paladin/Berserker) · **Marchand** (bonus d’échanges, taxes réduites, insights).
* Chaque classe : **stats de base**, **missions exclusives**, **évolutions** en sous-classes via **XP** + **ressources**.&#x20;

### Interface de jeu & navigation

#### 7) Lobby principal — 5 sections

* **Carte (Monde)** : exploration géolocalisée _(avec ses propres onglets d’exploration)_
* **Inventaire** : NFT & ressources (familiers, monnaies, skins, etc.)
* **Wallet** : porte-monnaie & **staking**
* **Marché** : marketplace du jeu + items joueurs (**agrégation Magic Eden**, listés ou non)
* **Profil** : stats, avatar, guildes
* **Notifications push** pour événements temps réel (ex. donjon proche).&#x20;

#### 8) Carte géolocalisée

* **Mapbox** + **OpenStreetMap** ; GPS temps réel du joueur. _(Section amorcée dans ta page ; détails à poursuivre dans “Gameplay > Navigation App / Exploration”.)_ .

### Progression & boucle de jeu

* **Onboarding** fluide (frictions réduites) pour novices **et** experts Web3.
* Progression reliée aux **rangs**, **classes**, **missions**, **économie** — **structurelle**, pas cosmétique. _(Détails dans onglets Gameplay/Progression.)_ .
