# Structure Économique

L’économie d’IKAI repose sur un **modèle circulaire**, où chaque ressource est produite, transformée, puis consommée dans le jeu. Rien n’est créé ex nihilo. La valeur provient de l’**activité réelle des joueurs**.

#### 1) Cycle Économique Global

```
Exploration → Récolte → Transformation → Échange → Utilisation → Progression → Nouvelle Exploration
```

| Étape                                                        | Qui agit             | Ce qui est généré                 | Où ça circule                |
| ------------------------------------------------------------ | -------------------- | --------------------------------- | ---------------------------- |
| Récolte (Monde / Donjons)                                    | Tous les joueurs     | Matériaux & Essences              | Inventaires personnels       |
| Production (Classe Agricole / Forge / Alchimie / Invocation) | Classes spécialisées | Objets, armes, potions, familiers | Marché régional              |
| Distribution                                                 | Marchands            | Mise en vente, arbitrage de prix  | Marketplace régionale        |
| Utilisation                                                  | Tous les joueurs     | Buffs, équipement, progression    | Consommation (Burn naturel)  |
| Progression                                                  | Tous les joueurs     | Montée en puissance               | Nouveau besoin de ressources |

Ce modèle **force la demande** et empêche la saturation.

***

#### 2) Interdépendance Directe (Sans spéculation)

| Classe          | Produit                  | Nécessaire à                  | Impact économique                                   |
| --------------- | ------------------------ | ----------------------------- | --------------------------------------------------- |
| **Agriculteur** | Nourriture (buffs)       | Tous les joueurs              | Consommation quotidienne → Demande continue         |
| **Forgeron**    | Armes / Armures / Skins  | Guerriers & tous explorateurs | Durabilité → renouvellement → demande cyclique      |
| **Alchimiste**  | Potions / Mana / Soin    | Tous les rôles de combat      | Système de récupération → consommation structurelle |
| **Invocateur**  | Insignes & familiers     | Tous joueurs                  | Progression & spécialisation → rareté maîtrisée     |
| **Marchand**    | Distribution & Arbitrage | Toute la région               | Régulation locale de prix → stabilité régionale     |

Aucune classe ne peut être autonome. Chacun **a besoin** des autres.

#### 4) Résultat

* La valeur **ne vient pas** de nouveaux entrants
* La valeur **ne se stocke pas**, elle circule
* La valeur **n’est jamais produite infiniment**
* Chaque joueur contribue naturellement à la stabilité
