# ⚡ Whitepaper IKAI

#### 1. Vision & Concept Live-to-Earn (L2E)

**Ikai** est une application mobile Web3 construite sur **Solana**, qui fusionne :

* Exploration **du monde réel**
* Mécaniques **RPG**
* Classes **interconnectées**
* **Économie circulaire décentralisée**
* Système **territorial géopolitique** basé sur les villes réelles

→ **Se déplacer, jouer, vivre** devient une **activité productive**.

Le modèle **L2E – Live-to-Earn** transforme l’énergie quotidienne en valeur :

* Marcher → XP
* Explorer → Loot
* Combattre → Ressources & NFT
* Participer → Gagner des tokens

Ikai crée un monde où l’activité réelle génère une économie durable.

#### 2. Expérience Utilisateur & Onboarding

**2.1 Splash Screen**

* Barre d’énergie **arc-en-ciel néon animée**
* Mascotte **Slime** (référence Isekai)
* **Auto-scan blockchain Solana**
* Chargement ultra rapide, optimisé mobile

**2.2 Connexion**

| Méthode                                | Avantages              |
| -------------------------------------- | ---------------------- |
| **Wallet Solana** (Phantom, Backpack…) | Connexion Web3 native  |
| **Email (OTP)**                        | Simple & instantané    |
| **Téléphone (OTP)**                    | Accessibilité maximale |

Si l’utilisateur n’a pas de wallet → **assistant intégré** (création/import en < 1 min).

**2.3 Sécurité / Récupération**

Ikai **innove contre l’oubli de seed** : La phrase secrète est **envoyée chiffrée par email** → récupérable même en cas de perte du téléphone. (Résout le problème des captures d’écran & post-it perdus.)

#### 3. Personnage & Identité

| Élément        | Description                                |
| -------------- | ------------------------------------------ |
| Genre          | **Homme / Femme / Androgyne**              |
| Avatar de base | **3 modèles** évolutifs                    |
| NFT Personnage | **Unique, non transférable, non vendable** |
| Rôle           | Identité du joueur dans tout l’écosystème  |

**3.1 Packs Partenariats (Burn NFT)**

* Si le joueur possède un NFT d’une **collection partenaire**
* Il peut **burn** pour obtenir un **NFT IKAI Partenaire** **transférable**
* Burn multiple = **NFT plus puissant + meilleure entrée**

**Exemple Dog :**

| Burn NFT | Récompense                              |
| -------- | --------------------------------------- |
| 1 NFT    | Pack 1 (valeur \~0.5 SOL)               |
| 8 NFT    | Pack 3 directement (sans payer le pack) |

#### 4. Packs d’Entrée (Starter Packs)

Tous les packs contiennent :

* NFT Personnage **évolutif & non transférable**
* **Objets NFT** (armes, skins, familiers) → **transférables**
* Ressources initiales (bronze → or)

#### 5. Système de Rangs & Transcendance

| Catégories                         | Description                          |
| ---------------------------------- | ------------------------------------ |
| **Rangs G → SSS**                  | Affectent gains, rareté, animations  |
| **Rangs Divins (12 + variations)** | Exclusifs événements / achat spécial |
| **Transcendance**                  | Permet d’upgrader **tout rang**      |
| **Coût d’upgrade**                 | → Augmente fortement selon le rang   |

**Plus haut le rang → Plus difficile la transcendance.**

#### 6. Classes & Économie Fonctionnelle

Ikai repose sur **12 classes interconnectées**, créant une **économie circulaire**.

| Classe      | Fonction           | Impact Économique           |
| ----------- | ------------------ | --------------------------- |
| Invocateur  | Familiers NFT      | Offre d’animaux rares       |
| Agriculteur | Ressources IRL     | Matières premières crafting |
| Guerrier    | Combats / Donjons  | Loot rare                   |
| Marchand    | Échanges optimisés | Fluidité du marché          |

Chaque classe :

* Possède **statistiques propres**
* Accède à **missions exclusives**
* Évolue en **sous-classes** via **XP + ressources**

→ **Aucune classe ne peut être autonome**. → La **coopération est obligatoire**.

#### 8. Progression, Énergie & Boucle de Jeu

**Énergie**

* 1 à 3 barres au départ
* Jusqu’à **24h maximum**
* Recharge **13h UTC**

**Boucle de Jeu**

* Marcher → XP + Loot
* Missions & Donjons → Récompenses
* Craft → Économie
* Marketplace → Liquidité
* Classe → Spécialisation
* Rang → Scaling

#### 9. Tokenomics

| Token      | Type                                       | Rôle                                 |
| ---------- | ------------------------------------------ | ------------------------------------ |
| **$IKAI**  | Utility Token                              | Achats, upgrades, staking            |
| **$xIKAI** | Token de stacking interne **non tradable** | Récompenses joueurs, valeur protégée |

* **0% vente privée**
* **0% pré-vente**
* **100% supply contrôlée par l’activité des joueurs**
* L’équipe ne gagne que sur :
  * Marketplace (6%)
  * Taxe RWA
  * Achats d’énergie / upgrades

→ **Anti-spéculation structurelle**
