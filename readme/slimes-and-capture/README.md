# Slimes & Capture

**Contexte** : Les slimes sont les créatures les plus communes dans l’univers d’IKAI.\
Ils apparaissent autour du joueur lors de ses déplacements, en temps réel.

***

#### Apparition des Slimes

* Apparition **continue**, liée à la **position GPS**.
* La **région**, le **continent**, le **biome** et certains **événements** influencent la **variété des slimes**.
* L’apparition **n’est pas influencée** par :
  * Le niveau du joueur
  * Son rang
  * Sa puissance
  * Sa fréquence de jeu

Un nouveau joueur peut croiser un slime rare.\
Un joueur avancé peut croiser un slime faible.\
La rencontre reste **organique et imprévisible.**

***

#### Combat Instantané

Le combat se résout par une comparaison directe :

```
Puissance du Joueur (ATK + DEF + bonus)
vs.
Puissance du Slime (valeur interne)
```

| Issue    | Effet                                                               |
| -------- | ------------------------------------------------------------------- |
| Victoire | Le slime peut être **capturé**                                      |
| Défaite  | Le joueur perd des **Points de Vie** (réparables via items / repos) |

***

#### Système d’Énergie : **Mana**

Chaque combat consomme du **Mana**, que l’on régénère :

* Par le **déplacement réel**
* Par le **temps**
* Par des **potions de mana** (non rentables pour farm token, mais utiles pour jouer)

| Type de Slime           | Coût en Mana |
| ----------------------- | ------------ |
| Petit                   | 1 min        |
| Normal                  | 2 min        |
| Moyen                   | 4 min        |
| Grand                   | 8 min        |
| Géant                   | 16 min       |
| **Rainbow** (très rare) | **32 min**   |

***

#### Capture & Fusion&#x20;

Après une victoire, le joueur peut :

* **Capturer** le slime
* Ou le **laisser disparaître**

La capture nécessite une **Pierre de Capture** (craft ou loot).

Une fois capturé :

* Le slime est **stocké** dans le **Slime Book** (collection personnelle).
* Les slimes peuvent être **fusionnés** entre eux :
  * Un slime **absorbe** un autre → **gain d’XP**
  * Objectif : **completer et améliorer la collection**



C’est une **collection stratégique** destinée **à évoluer plus tard**.

***

#### Résumé&#x20;

* Les slimes sont présents partout dans le monde réel.
* Les combats sont rapides et basés sur les statistiques.
* La capture permet de constituer une **collection vivante**.
* La fusion permet de **faire grandir** certains slimes.
* Leur **rôle avancé** sera révélé **plus tard** dans la progression.
