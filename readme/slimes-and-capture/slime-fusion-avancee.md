# Slime Fusion Avancée

Cette mécanique n’est **pas disponible au lancement**.\
Elle sera introduite **lors d’une saison ou extension majeure.**

***

### Fusion Slime → Armure Vivante

À un certain stade d’évolution, un **slime suffisamment développé** peut **fusionner avec le joueur**.

Le slime devient alors :

* **Une extension du corps**
* **Une armure vivante**
* **Un amplificateur de capacités**

Visuellement : le joueur **porte** le slime, qui **modifie sa silhouette, ses effets et ses compétences.**

***

#### Effets selon le type de Slime

| Type élémentaire      | Transformation Prévue   | Exemple de capacité              |
| --------------------- | ----------------------- | -------------------------------- |
| Eau                   | Gaine fluide            | Jet haute pression / Hydrocannon |
| Feu                   | Carapace incandescente  | Projection de flammes            |
| Terre                 | Forme lourde / blindage | Bouclier naturel / résistance    |
| Acide                 | Structure semi-liquide  | Dissolution / corrosion          |
| Arc-en-ciel (Rainbow) | Morphologie instable    | Compétences **uniques**          |

Ce système **crée de la diversité** sans créer de **déséquilibre** :\
Les compétences **ne dépassent pas** la logique de progression déjà existante.

***

#### Ce que permet ce système

* Longévité du gameplay
* Identité visuelle unique
* Personnalisation avancée
* **Progression méritée**, pas achetée

Ce n’est **pas** du Pay-to-Win.\
C’est du **Play-to-Unlock**.

***

#### Résumé&#x20;

* Le slime est la **mécanique centrale cachée** du jeu.
* Elle fait passer le joueur de **simple avatar** → à **entité vivante liée à son environnement**.
